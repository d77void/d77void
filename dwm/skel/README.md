## Hello

Thank you for trying d77void.

To install use Calamares from the menu.

In case it won't start, open a terminal and type:

```
sudo -E calamares
```

## Keybinds

alt + shift + return -> terminal

alt + shift + q -> logout

alt + shift + c -> close window

alt + t -> tile mode

alt + f -> floating mode

alt + m -> focus mode

alt + p -> dmenu

alt + r -> gmrun

alt + s -> scratchpad

alt + x -> powermenu

alt + w -> brave

super + b -> swap wallpaper

super + d -> rofi

super + f -> pcmanfm

super + l -> slock

super + m -> geary

super + p -> scrot

super + Escape -> reload sxhkd config

## Hacking

To add/remove autostart programs edit the file ~/.dwm/autostart.sh

Have fun!
