Thank you for trying d77void. This ISO qith Qtile is hybrid (X11/Wayland).


To install use Calamares from the menu.

In case it won't start, open a terminal and type:

```
sudo -E calamares
```

In Wayland, tray, notifications and locker don't work properly.

## Keybinds

super + return -> terminal

super + b -> swap wallpaper

super + a -> menu (x11 and wayland differ this menu)

super + d -> file manager

super + w -> browser

super+ m -> mail

super+ n -> menu (x11 and wayland differ this second menu) 

super + i -> swap keyboard layout

super + tab -> swap layout

super + r -> run

super + x -> powermenu

super + ctrl + q -> logout

super + ctrl + r -> reload config

super + q -> close window

Have fun!
