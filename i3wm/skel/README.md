To install use Calamares from the menu.

In case it won't start, open a terminal and type:

```
sudo -E calamares
```

## Keybinds

alt + return -> terminal

alt + shift + e -> logout

alt + shift + q -> close window

alt + d -> rofi

alt + x -> powermenu

alt + t -> lock screen

Have fun!
