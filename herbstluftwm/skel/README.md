## Hello

Thank you for trying d77void.

To install use Calamares from the menu.

In case it won't start, open a terminal and type:

```
sudo -E calamares
```

## general Keybinds

alt + shift + return -> terminal

alt + shift + q -> logout

alt + shift + c -> close window

alt + shift + r -> reload configuration


## d77 Keybinds

alt + a -> scratchpad

alt + b -> swap wallpaper

alt + d -> rofi

alt + e -> mousepad

alt + m -> mail

alt + t -> lock screen

alt + w -> brave

alt + x -> powermenu

alt + z -> yakuake alike scratchpad

super + f -> pcmanfm

super + j -> jgmenu

super + p -> scrot

super + r -> gmrun


Have fun!
