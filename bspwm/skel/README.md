## Hello

Thank you for trying d77void.

To install use Calamares from the menu.

In case it won't start, open a terminal and type:

```
sudo -E calamares
```

## Keybinds

super + return -> terminal

super + w -> close active window

super + d -> menu

super + space -> menu2

super + x -> powermenu

super + h -> navigate left window

super + l -> navigate right window

super + j -> navigate down window

super + k -> navigate up window

super + alt + q -> exit bspwm

Have fun!
