## Installer

To install use Calamares from the menu.

In case it won't start, open a terminal and type:

```
sudo -E calamares
```

## Keybinds

super + shift + return -> terminal

super + shift + e -> logout

super + q -> close window

super + d -> menu1

super + m -> mail

super + n -> menu2

super + o -> gmrun

super + t -> lock

super + w -> web browser

super + x -> powermenu

super + shift + f -> file manager


# River Tweaks

To alter the autostart programs, add/swap keybinds and keyboard layout configure ~/.config/river/init ; the file is self explanatory and with several examples already configured.

Have fun!
